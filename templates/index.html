South Park Canadian GTA-Style Game

Your game is running in a separate window. To view and interact with it, click the VNC button in the Replit sidebar.

How to Play

  • WASD: Move character
  • E: Enter/exit vehicle or talk to nearby NPCs
  • SPACE: Shoot (if you have a weapon)
  • Arrow Keys: Used for cheat code sequences
  • ESC or P: Pause game
  • Ctrl+Q: Exit game (PC only)

Tips

The game can be played with both keyboard controls and touch controls on mobile devices.

Press F3 to toggle debug information display.

Auto-control is enabled by default to help when keyboard input isn't working properly. Look for the auto-control status display in the top right corner.

New Features

  • Cheat Codes: Find secret cheat codes by talking to NPCs and experimenting with arrow key sequences.
  • Dialogue System: Press E near pedestrians to talk with them and get hints about cheat codes.
  • Side Activities: Discover side activities throughout the city, including illegal taxi service, garbage truck races, and underground fight clubs.
  • Escalating Events: Stay in one area for too long and watch as chaotic events naturally escalate around you!

If the game doesn't appear, try restarting the workflow or check the console for errors.
